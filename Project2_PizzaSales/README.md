# Pizza Sales Performance Dashboard (Power BI + PostgreSQL)

## Overview
This project analyzes pizza sales data to uncover customer preferences, sales trends, and product performance. The dashboard provides KPIs, time-based trends, and category breakdowns to support business decision-making.

## Tech Stack
- PostgreSQL (Data source, SQL queries for validation)
- Power BI (Data modeling, DAX, Visualizations)

## Key Features
- Designed KPIs (Revenue, Orders, Avg Order Value, Total Pizzas Sold, Avg Pizzas per Order) and built interactive visuals (daily/monthly trends, category breakdowns, funnel chart by category, sales distribution by size).
- Applied SQL queries for data extraction and validation, and DAX measures for time intelligence and dynamic reporting.
- Delivered business insights into busiest days, most popular categories, and customer preferences by pizza size.

## Business Insights
- Fridays had the highest order volumes, showing peak demand.
- Supreme and Chicken pizzas were top categories by revenue.
- Medium and X-Large pizzas dominated sales by size.
- Seasonal demand fluctuations observed across months.
