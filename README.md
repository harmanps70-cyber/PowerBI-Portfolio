# Power BI Portfolio Projects

This repository showcases end-to-end BI projects built in Power BI, SQL, and DAX. Each project demonstrates KPI design, data modeling, and dashboard storytelling for business insights.

## Projects Included
- **Company ABC Performance Dashboard (Excel → Power BI)**  
  Advanced DAX, time intelligence, dynamic measure switching, conditional formatting.

- **Pizza Sales Performance Dashboard (PostgreSQL → Power BI)**  
  End-to-end database integration, KPI design, SQL validation, interactive visuals.

## Tech Stack
- Power BI (Data modeling, DAX, Visualizations)
- SQL (PostgreSQL, validation queries)
- Excel (Data source for Project 1)
